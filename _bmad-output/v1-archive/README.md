# v1 Archive (Next.js PWA)

This folder contains artifacts from the v1 implementation that was deprecated on 2026-01-26.

## What Was v1?

- **Platform:** Progressive Web App (PWA)
- **Framework:** Next.js 14
- **Auth:** Supabase Auth
- **Database:** Supabase PostgreSQL
- **Backend:** Supabase Edge Functions
- **Styling:** Tailwind CSS
- **State:** TanStack Query + Zustand
- **Offline:** IndexedDB + Service Workers

## Why Archived?

The project pivoted to a native mobile app (v2) with Expo + Convex + Clerk for better performance, real-time features, and platform-native experience.
